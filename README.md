**众包平台（CrowdSourcing Platform）**是一个专门为当前软件需求旺盛的当今应运而生的平台。平台集项目交易、项目托管、代码检测、需求分析等功能于一体，为项目开发者与项目需求者都提供一个良好的环境。

-**安全可靠**

-**功能丰富**

-**深度整合**



## 本项目技术简介（前端）

-**后端**： [点击前往](https://github.com/Lab409-SSE-Tongji/crowdsourcing-mart-api)
